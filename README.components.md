### 组件说明文档
以下组件可在 @/demo下查看用法
```
├─ @/components    // 组件库
│ ├── BackToTop    // 返回顶部
│ ├── Breadcrumb    // 面包屑
│ ├── Charts    // 图表 
│ ├── DndList    // 列表拖拽
│ ├── DragSelect    // 拖拽Select
│ ├── Dropzone    // 拖拽上传
│ ├── ErrorLog    // 错误日志弹窗
│ ├── Hamburger    // 控制左边菜单栏缩放
│ ├── HeaderSearch    // 全局搜索
│ ├── ImageCropper    // 上传组件
│ ├── JsonEditor    // json编辑器
│ ├── Kanban    // 可拖拽看板
│ ├── MarkdownEditor    // Markdown编辑器
│ ├── MDinput    // 输入框组件
│ ├── Pagination    // 分页组件
│ ├── PanThumb    // 头像旋转动画组件
│ ├── RightPanel    // 右侧菜单栏
│ ├── Screenfull    // 全屏显示
│ ├── Share    // 收缩菜单栏
│ ├── SizeSelect    // 布局大小
│ ├── Sticky    // 粘性header组件, 当页面滚动到预设的位置会吸附在顶部
│ ├── SvgIcon    // 
│ ├── TextHoverEffect    // 文字动画效果
│ ├── ThemePicker    // 主题颜色
│ ├── Tinymce    // 富文本编辑器
│ ├── Upload    // 图片上传
│ └── UploadExcel    // 上传表格
```
